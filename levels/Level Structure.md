Version
Version reffers to format versioning.

N
N reffers to grid size. (ex. 10x10)

K
K reffers to globes per constraint group.

Level
Level reffers to level number.

Difficulty
Difficulty reffers to the difficulty of a level.

Unique
Unique reffers to if there are multiple solutions to the level.
True reffers to only one solution and false reffers to multiple.

Regions
Regions define the board partition.
